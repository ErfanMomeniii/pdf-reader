# PDF Reader

A fast, lightweight PDF viewer for macOS, Windows, and Linux.

## Download

Download the latest version for your operating system:

| Platform | Download |
|----------|----------|
| macOS | [PDF-Reader.dmg](../../releases/latest) |
| Windows | [PDF-Reader-Setup.exe](../../releases/latest) |
| Linux | [PDF-Reader.AppImage](../../releases/latest) |

Or visit the [Releases](../../releases) page for all versions.

## Features

- Fast PDF rendering with smooth scrolling
- Zoom in/out and fit to width
- Page navigation
- Drag and drop PDF files
- Recent files list
- Dark theme

## Keyboard Shortcuts

| Action | macOS | Windows/Linux |
|--------|-------|---------------|
| Open File | `Cmd + O` | `Ctrl + O` |
| Close File | `Cmd + W` | `Ctrl + W` |
| Zoom In | `Cmd + =` | `Ctrl + =` |
| Zoom Out | `Cmd + -` | `Ctrl + -` |
| Next Page | `Cmd + ]` | `Ctrl + ]` |
| Previous Page | `Cmd + [` | `Ctrl + [` |

## System Requirements

- **macOS**: 10.13 or later
- **Windows**: Windows 10 or later
- **Linux**: WebKitGTK 4.1+
